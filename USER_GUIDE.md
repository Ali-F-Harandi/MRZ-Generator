# User Guide

## Getting Started
Welcome to the MRZ Generator. This tool allows you to create valid Machine Readable Zone (MRZ) codes for official travel documents.

## Supported Documents (Coming Soon)
- **Passports**: Standard ICAO 9303 TD3 format.
- **ID Cards**: National ID formats (TD1, TD2).
- **Visas**: Standard Visa formats.

## How to Use
1. **Select Document Type**: Choose the type of document you want to generate.
2. **Enter Details**: Fill in the required personal information (Name, Passport Number, Nationality, etc.).
3. **View Result**: The MRZ code will appear automatically at the bottom.
4. **Copy/Download**: Use the action buttons to copy the text or save it.

*Note: This is a privacy-focused tool. No data entered here is sent to any server.*
